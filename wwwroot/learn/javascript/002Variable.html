variable JavaScript

  • variable are used to store information to be referenced & manipulated in a computer program.
  • In programming, a variable is a value that can be changed depending on the conditions or information being passed to the program.
  • Before you use a variable in a JavaScript program, you must declare it. Variables are declared with the var keyword

variable JavaScript

  • Variables can be thought of as named containers.
  • You can place data into these containers and then refer to the data simply by naming the container.
  • You can also declare multiple variables with the same var keyword

JavaScript "document.write"

  • One of the most basic JavaScript commands is document. write. This simply prints the specified text to the page.
  • To print text literally, enter the text in single quote marks inside parentheses
  • E.g.:- document.write('Hello World!');
  • The code above will cause the phrase "Hello World!" to appear on the page

JavaScript "document.write"

  • we can also use document.write to print variables.

    • Enter the variable name without quotes.

    e.g. var myText = "Hello again";

    document.write(myText);

How to use it? 


						<html>
						<body>
						<script>
						var firstNumber = 5;// firstNumber stores the value 5
						var secondNumber = 6;//secondNumber stores the value 6
						var total = firstNumber + secondNumber;//total stores the value 11
						document.write(total)//To print Variable
						</script>
						</body>
						</html>

JavaScript Alert

  • A JavaScript alert is a simple window containing a message.

  • An alert box is often used if you want to make sure information comes through to the user.

  • The alert box takes the focus away from the current window, and forces the browser to read the message.

  • E.g. alert("Hello world!");

    • How to use it? 

    
						<html>
						<body>
						<script>
						var firstNumber = 5;// firstNumber stores the value 5
						var secondNumber = 6;//secondNumber stores the value 6
						var total = firstNumber + secondNumber;//total stores the value 11
						document.write(total)//To print Variable total 11
						alert(total)// to alert variable total 11
						</script>
						</body>
						</html>

    prompt() Method

    • The prompt() method displays a dialog box that prompts the visitor for input.
    • The prompt() method returns the input value if the user clicks "OK". If the user clicks "cancel" the method returns null.
    • The method accepts two arguments, but only one of them is necessary for the function to work:

      • prompt(text, defaultText)

    How to use it? 

    
										<html>
										<body>
										<script>
										var firstNumber = prompt("enter a number");//dialog box for input
										var secondNumber =prompt("enter a number");//dialog box for input
										var number1=parseInt(firstNumber);
										var number2=parseInt(secondNumber);
										var total = number1 + number2;//total stores the value 11
										console.log(total)//to display values in the debugger window
										document.write(total)//To print Variable total 11
										alert(total)// to alert variable total 11
										</script>
										</body>
										</html>
